* Introduction

~early-init.el~ was introduced in Emacs 27 and is loaded before the regular ~init.el~.
Because it runs before GUI and package system initialization, it is suitable for settings such as:

- Frame and window appearance
- Package initialization control
- Startup flicker prevention

* early-init.el

** header

#+begin_src emacs-lisp
  ;;; package --- Early initialization file -*- lexical-binding: t; -*-
  ;;; Commentary:
  ;;; Code:
#+end_src

** Change the Storage Location for Custom Variables

Store custom variables configured with ~M-x customize~ and similar commands in a separate file from ~init.el~.
This separates automatically generated custom settings from manually written configuration.

#+begin_src emacs-lisp
  (setq custom-file (locate-user-emacs-file "custom.el"))
  (when (file-exists-p (expand-file-name custom-file))
    (load-file (expand-file-name custom-file)))
#+end_src

** Maximize the Frame

Maximize the Emacs frame at startup.
Setting this in ~early-init.el~ avoids the flicker caused by a small window appearing briefly before being maximized.

#+begin_src emacs-lisp
  (push '(fullscreen . maximized) default-frame-alist)
#+end_src

** Hide the Scroll Bar

Hide the GUI scroll bar.

#+begin_src emacs-lisp
  (scroll-bar-mode -1)
#+end_src

** Hide the Menu Bar

Hide the menu bar at the top of the frame (File, Edit, ...).

#+begin_src emacs-lisp
  (menu-bar-mode -1)
#+end_src

** Hide the Tool Bar

Hide the GUI tool bar.

#+begin_src emacs-lisp
  (tool-bar-mode -1)
#+end_src

** footer

#+begin_src emacs-lisp
  (provide 'early-init)
  ;;; early-init.el ends here
#+end_src
