* Introduction

This file describes my Emacs configuration in literate programming format.
It uses ~org-babel-load-file~ to generate ~init.el~.

** Features

- *Modern package management*: Uses ~use-package~ and ~package.el~
- *Fast completion UI*: Combines Corfu, Vertico, Consult, and Embark
- *Tree-sitter support*: Accurate syntax highlighting and code analysis
- *LSP support*: Multi-language support through ~lsp-mode~
- *Multi-OS support*: Supports Windows and Linux (macOS is untested)

** Usage

1. Place this file at ~~/.emacs.d/init.org~.
2. Start Emacs and run ~M-x org-babel-load-file~.
3. The generated ~init.el~ will be loaded from the next startup.

** Required External Tools

- ~cmigemo~: Romaji-based Japanese search (optional)
- ~ripgrep~: Fast file search (recommended)
- ~fd~: Fast file search (recommended)
- Language parsers for Tree-sitter

*** [[https://github.com/minad/cape][cape]] (Completion At Point Extensions)

This package collection extends Emacs's standard completion interface, ~completion-at-point-functions~ (CAPF).
It makes it easy to add and manage completion sources, which are functions that provide candidates, for completion UIs such as Corfu and Company.

#+begin_src emacs-lisp
  (use-package cape
    :init
    (with-eval-after-load 'lsp-completion
      (advice-add 'lsp-completion-at-point
                  :around #'cape-wrap-noninterruptible))

    :config
    (require 'cl-lib)
    (autoload 'cape-keyword "cape-keyword" nil t)
    (autoload 'tempel-complete "tempel" nil t)

    (defconst my/cape-extra-capfs
      '(cape-dabbrev
        cape-keyword
        cape-elisp-block
        tempel-complete))

    (defvar-local my/cape--base-capfs nil)

    (defun my/cape--managed-capf-p (capf)
      (or (memq capf '(t cape-file my/cape-super-capf))
          (memq capf my/cape-extra-capfs)))

    (defun my/cape--remember-capfs ()
      (let ((capfs (cl-remove-if #'my/cape--managed-capf-p
                                  completion-at-point-functions)))
        (when capfs
          (setq-local my/cape--base-capfs
                      (cl-delete-duplicates
                       (append capfs my/cape--base-capfs)
                       :test #'eq)))))

    (defun my/cape--super-capf ()
      (apply #'cape-wrap-super
             (append my/cape--base-capfs my/cape-extra-capfs)))

    (defalias 'my/cape-super-capf
      (cape-capf-sort
       (cape-capf-nonexclusive
        (cape-capf-buster #'my/cape--super-capf))))

    (defun my/cape-setup-capfs ()
      (unless (minibufferp)
        (my/cape--remember-capfs)
        (setq-local completion-at-point-functions
                    '(cape-file my/cape-super-capf t))))

    (dolist (hook '(after-change-major-mode-hook
                    corfu-mode-hook
                    eglot-managed-mode-hook
                    lsp-completion-mode-hook))
      (add-hook hook #'my/cape-setup-capfs)))
#+end_src

** Other Convenient Functions

*** Convert Data Extracted from a Database into an IN Clause

Read the selected text line by line, convert it to a SQL IN clause format, such as IN ('A', 'B', 'C'), and replace the selection.
Duplicate lines are removed. This is useful for formatting ID lists copied from a database into SQL queries.

#+begin_src emacs-lisp
  (defun my/replace-region-with-in-clause (start end)
    "Convert the selected region from START to END into a SQL IN clause and replace the original selection."
    (interactive "r") ; Callable when the region is active.
    (let* ((region-content (buffer-substring-no-properties start end)) ; Get the selected text.
           (records (split-string region-content "\n" t)) ; Split into a list by newlines, ignoring empty lines.
           (unique-records (delete-dups records))        ; Remove duplicates.
           ;; Wrap each record in single quotes.
           (quoted-records (mapcar (lambda (record) (format "'%s'" record)) unique-records))
           ;; Generate the IN clause string.
           (in-clause (format "IN (%s)" (mapconcat 'identity quoted-records ", "))))
      (delete-region start end) ; Delete the original selection.
      (insert in-clause)))      ; Insert the generated IN clause.
#+end_src
